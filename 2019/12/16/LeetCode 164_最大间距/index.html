LeetCode 164_最大间距

因实验室要求，所以写一下每周算法中的算法题，题是LeetCode中的第164题，最大间距。

题目描述和相应的样例

给定一个无序的数组，找出数组在排序之后，相邻元素之间最大的差值。

如果数组元素个数小于 2，则返回 0。

示例 1:

输入: [3,6,9,1]
输出: 3
解释: 排序后的数组是 [1,3,6,9], 其中相邻元素 (3,6) 和 (6,9) 之间都存在最大差值 3。
示例 2:

输入: [10]
输出: 0
解释: 数组元素个数小于 2，因此返回 0。
说明:

你可以假设数组中所有元素都是非负整数，且数值在 32 位有符号整数范围内。
请尝试在线性时间复杂度和空间复杂度的条件下解决此问题。

来源：力扣（LeetCode）
链接：https://leetcode-cn.com/problems/maximum-gap

思路介绍

最简单的方法我就不再叙述了，就是先排序，再依次比较相邻的元素。

这里我们主要讲一下最优解，最优解用到了桶排序的思想，关于各种排序，我在查询资料的时候，找到了一篇我认为写的特别好的文章，这里贴一下链接，这或许是东半球讲十大排序算法最好的一篇文章。

为什么说用到了桶排序的思想，此题给定的是无序数组，并且求的是最大间距。我们能将所有数据利用桶排序的思想，放在桶里。比如有10个数，他们的范围是1 ~ 100。我们就创建10个桶，第一个桶存放1 ~ 10，第二个桶存放11 ~ 20，以此类推。把所有的数据放在相应的桶中，现在问题就变得简单了起来。最大间距现在在只可能是两种情况，第一种情况是一个桶内的最小值和最大值的间距，第二种情况是两个相邻的桶，后一个桶的最小值和前一个桶的最大值的间距。

思路简单叙述的话就是如上这些，具体代码我就不贴上来了，如有关于代码或者思路上的问题，欢迎联系作者交流。
